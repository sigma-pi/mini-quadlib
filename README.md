# Introduction
A mini library for quadrotor control.

# Usage

## Source Build

```bash
sudo apt update
sudo apt install cmake build-essential

cmake -B build && cmake --build build

```

## To Contribute

**Pull request types:**
- `fix/` - bug fix branch
- `feature/` - new feature branch
- `hotfix/` - urgent fix branch
- `chore/` - maintenance/cleanup branch

```bash
git checkout -b feature/[feature names]
git commit -m "[feature descriptions]"
git push origin feature/[feature names]
```
Create pull request after pushing your changes to the new branch "feature/[feature names]".
